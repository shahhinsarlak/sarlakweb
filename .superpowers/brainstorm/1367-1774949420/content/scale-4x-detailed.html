1× vs 4× — same feature, more ASCII density

The 4× version is a fully redrawn, denser art piece. Same character — far more detail.

How this works in the game: each customisation slot has two ASCII art definitions — a compact version (rendered in the room at normal font size) and a dense version (rendered in the character creator selection panel at the same font size but far more lines/characters). Same ASCII renderer, different art assets. The dense version makes the creator feel like a proper character sheet.
